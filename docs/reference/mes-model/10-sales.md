# 10 — 销售与出货管理

> **实体数：** 5 | **流程数：** 2  
> **覆盖：** 销售订单、出货管理、送货

---

## 数据实体

| 实体 | 说明 |
|------|------|
| 客户数据 | 客户主数据 |
| 销售订单数据 | 销售订单信息 |
| 出货单数据 | 出货单/拣货单 |
| 送货单 | 物流送货单 |
| 销售数据 | 销售数据汇总 |

---

## 业务流程

| 流程 | 说明 | 关联主要实体 |
|------|------|-------------|
| 销售订单维护 | 销售订单管理 | 客户数据、销售订单数据 |
| 打印出货单 | 出货单打印 | 出货单数据 |
| 成品出库 | 成品从仓库出库 | 销售数据、库存数据(销售) |
| 装车采集 | 装车信息采集 | 装车记录 |
| 生成送货单 | 送货单生成 | 送货单 |

---

## 关联流程

```
销售订单 → 生产工单（按订单生产）
         → 成品出库（按订单发货）
         → 送货单（物流配送）

销售退货：
  客户退货 → 生成销售退货单据 → 退货入库 → OQC检验
```
