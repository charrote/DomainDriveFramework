# 01 — 组织与人员管理

> **实体数：** 19 | **流程数：** 9  
> **覆盖：** 组织架构、人员信息、班次管理、技能管理、权限账号

---

## 数据实体

### 组织架构
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 组织架构 | 企业组织层级定义 | 架构编码、架构名称、上级架构 |
| 组织层级 | 层级关系表 | 层级编码、层级级别 |
| 员工组 | 人员分组（如：白班组、夜班组） | 组编码、组名称 |
| 库存组织 | 库存管理组织单元 | 组织编码、组织名称 |

### 人员
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 人员 | 作业人员基础信息 | 工号、姓名、所属组织、技能等级 |
| 帐号 | 系统登录账号 | 用户名、密码、关联人员 |
| 人员与组织层级关系 | 人员归属关系 | 人员、组织层级、生效日期 |
| 人员与班组关系 | 人员所属班组 | 人员、班组、班次 |
| 人员技能 | 人员技能标签 | 人员、技能编码、技能等级 |
| 人员与资源关系 | 人员可操作的设备/资源 | 人员、资源 |
| 人员与工序关系 | 人员可执行工序 | 人员、工序、资质有效期 |

### 班次/日历
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 班组 | 生产班组 | 班组编码、班组名称 |
| 班制 | 排班制度（两班倒/三班倒等） | 班制编码、班制名称 |
| 班次 | 具体班次时间段 | 班次编码、开始时间、结束时间、所属班制 |
| 休息时间 | 班次内休息时段 | 开始时间、结束时间 |
| 资源日历 | 设备/人员的工作日历 | 资源、日期、可用时段 |
| 工厂日历 | 工厂级日历（工作日/节假日） | 日期、是否工作日 |

### 其他
| 实体 | 说明 |
|------|------|
| 资源层级 | 资源分类层级（用于设备和人员分类） |
| 资源模型 | 资源属性模型定义 |

---

## 业务流程

| 流程 | 说明 | 关联主要实体 |
|------|------|-------------|
| 组织架构维护 | 维护企业组织层级 | 组织架构、组织层级 |
| 组织层级维护 | 设置组织层级级别 | 组织层级 |
| 人员与组织层级关系维护 | 分配人员到组织单元 | 人员、组织层级 |
| 人员管理 | 人员信息管理 + 技能管理 | 人员、人员技能、帐号 |
| 人员分组 | 将人员分组管理 | 员工组 |
| 班组维护 | 班组信息维护 | 班组 |
| 班制维护 | 班制信息的增删改 | 班制 |
| 资源日历维护 | 设置设备和人员的可用日历 | 资源日历、工厂日历 |
| 人员与资源关系维护 | 分配人员可操作的设备 | 人员、资源 |
| 人员与工序关系维护 | 分配人员可执行的工序 | 人员、工序 |
| 人员技能维护 | 人员技能标签管理 | 人员技能 |

---

## 关键关系

```
工厂 ──→ 车间 ──→ 产线 ──→ 工位
  ↑         ↑        ↑        ↑
  │         │        │        │
人员 ──→ 班组 ──→ 班次 ──→ 资源日历

人员 ──→ 帐号（一个人员一个系统账号）
人员 ──→ 技能（多对多）
人员 ──→ 资源/设备（多对多，可操作哪些设备）
人员 ──→ 工序（多对多，可执行哪些工序）
```
