## 核心业务流程

```
采购入库流程：
  PO发布 → PO接收 → 接收 → 检验交接(IQC) → 入库交接 → 上架

生产执行流程：
  工单管理 → 工单下载 → 上线 → 上料 → 产品装配 
  → PQC → 下线 → 包装 → 包装复核 → 打印装箱单 → 成品出库

品质管控流程：
  IQC报检 → 来料检验 → (复判/返修) 
  PQC(制程检验) → 复判 → 维修/返修 
  OQC(成品检验) → 等级判定

物料流转：
  生成领料单 → 拣配 → 拣料 → 发料 → 上料 → 拉式倒冲/包材倒冲
  → 退料 → 退料标签打印
```

## 基础数据分类支撑

```
基础数据层贯穿所有领域：
  ├── 编码规则（条码/工单/质检单/设备等所有编号生成）
  ├── 单位管理（物料单位、单位换算）
  ├── 打印模板（标签、装箱单、出货单）
  ├── 日历管理（工厂日历、资源日历、班次/班制）
  └── 组织架构（工厂→车间→产线→工位）
```
