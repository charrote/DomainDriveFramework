# 03 — 物料与主数据

> **实体数：** 28 | **流程数：** 13  
> **覆盖：** 物料主数据、物料分类体系、物料属性、物料包装规则、基础编码

---

## 数据实体

### 物料主数据
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 物料 | 物料主数据核心 | 物料编码、物料名称、规格型号、分类、基本单位、物料类型（原材料/半成品/成品/辅料/包材）、状态 |
| 物料属性 | 物料的动态扩展属性 | 物料、属性名称、属性值 |
| 物料属性值 | 属性可选值字典 | 属性、可选值 |
| 物料通用属性 | 物料通用属性模板 | 属性名称、属性类型 |

### 物料分类
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 物料大类 | 一级分类 | 大类编码、大类名称 |
| 物料中类 | 二级分类 | 中类编码、中类名称、所属大类 |
| 物料小类 | 三级分类 | 小类编码、小类名称、所属中类 |

### 物料包装与单位
| 实体 | 说明 |
|------|------|
| 物料包装规则 | 物料的包装层级与数量关系（如：1箱=12个） |
| 单位 | 基本单位字典（个/kg/米/升/箱） |
| 物料单位转换关系 | 物料不同包装单位间换算关系 |

### 物料检验关联
| 实体 | 说明 |
|------|------|
| 物料检验项目 | 物料需执行的检验项目 |
| 免检物料清单 | 免于来料检验的物料白名单 |

## 分类体系

```
物料分类（三级）：
  物料大类（如：原材料/电子料/五金件/包材）
    └── 物料中类（如：五金件-螺丝/五金件-弹簧）
          └── 物料小类（如：M6螺丝/不锈钢弹簧）

物料属性（可扩展）：
  如：颜色、材质、硬度、耐温等级、环保等级...
```
