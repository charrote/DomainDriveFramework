# 04 — 工艺与 BOM 管理

> **实体数：** 19 | **流程数：** 12  
> **覆盖：** 工艺路线、工序定义、BOM结构、产品配置

---

## 数据实体

### BOM 体系
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 产品BOM | 产品物料清单主表 | BOM编码、产品编码、版本号、状态 |
| 工序BOM定义 | BOM中某道工序关联的物料 | BOM、工序、物料、用量 |
| 工序BOM | 工序级BOM（工单执行时使用） | 工单、工序、物料、用量 |
| 工单BOM | 工单级BOM（从产品BOM复制） | 工单、物料、需求数量、已消耗数量 |

### 工艺体系
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 工艺路线 | 产品工艺路线主表 | 路线编码、产品编码、版本号、总标准工时、状态 |
| 工序 | 工序定义 | 工序编码、工序名称、标准工时、所需设备类型 |
| 采集步骤 | 工序中的采集项定义（如：温度/压力/尺寸） | 工序、采集项名称、数据类型、规格上下限 |
| 工序与缺陷关系 | 工序可能产生的缺陷类型 | 工序、缺陷、缺陷等级 |

### 产品配置
| 实体 | 说明 |
|------|------|
| 产品工艺路线配置 | 产品与工艺路线的关联（一个产品可多条路线） |
| 产线工艺路线配置 | 产线与工艺路线的关联（一条产线可执行哪些路线） |
| 产品族 | 产品系列分组 |
| 产品分类 | 产品分类体系 |

---

## 业务流程

| 流程 | 说明 |
|------|------|
| 产品BOM维护 | BOM结构定义与维护 |
| 下载产品BOM | 从ERP同步BOM到MES |
| 工艺路线维护 | 工艺路线主数据维护 |
| 工序维护 | 工序定义维护 |
| 产品工艺路线设置 | 为产品分配工艺路线 |
| 产线工艺路线设置 | 为产线分配可执行的工艺路线 |
| 产品分类维护 | 产品分类体系的维护 |
| 产品族维护 | 产品系列分组的维护 |
| 维护工单工艺路线 | 工单执行时的路线调整 |
| 工单下载 | 从ERP同步工单到MES |

---

## 关键关系

```
产品 ──→ 产品BOM ──→ BOM明细（多级结构）
                     └── 工序BOM定义（关联工序和物料）

产品 ──→ 工艺路线 ──→ 工序（有序排列，每道工序可关联设备类型）
       └── 产品工艺路线配置（一个产品可指定默认路线）

产线 ──→ 产线工艺路线配置（产线可执行哪些路线）

工单 ──→ 工单BOM（从产品BOM复制生成）
     └── 工单工艺路线（从工艺路线复制生成，可微调）
```
