# 05 — 工单与生产执行

> **实体数：** 13 | **流程数：** 11  
> **覆盖：** 工单管理、上料/下料、装配执行、包装下线、容器管理

---

## 数据实体

### 工单
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 工单 | 生产工单核心 | 工单编号、产品编码、计划数量、完成数量、报废数量、状态、优先级、计划开始/结束、实际开工/完工、工艺路线、产线、班次 |
| 工单属性 | 工单的动态扩展属性 | 工单、属性名、属性值 |
| 工单BOM | 工单的物料清单 | 工单、物料编码、物料名称、需求数量、已消耗数量 |
| 工单工艺路线 | 工单执行的工序序列 | 工单、工序、序号 |
| 制造计划 | 生产排程计划 | 计划编码、产线、工单、计划时间 |

### 生产执行
| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 工单上线数 | 工单已投入到产线的数量 | 工单、累计上线数 |
| 工单完成数 | 工单已完成的数量 | 工单、累计完成数 |
| 工单实际上线时间 | 工单上线的时间记录 | 工单、上线时间、数量 |
| 上料记录 | 物料被投放到工位/设备的记录 | 工单、工序、物料、数量、时间、操作人 |
| 下料记录 | 半成品/成品从工位取下的记录 | 工单、工序、物料、数量、时间、操作人 |
| 产品装配记录 | 装配工序的产出入记录 | 工单、工序、装配物料、产出数量、操作人、时间 |
| 过站记录 | 产品经过某工序的扫描记录 | 产品条码、工序、时间、操作人 |

### 容器/配送
| 实体 | 说明 |
|------|------|
| 容器 | 料箱/料架/托盘等装载容器 |
| 容器与物料关系 | 容器当前装载的物料 |
| 清空的容器 | 已清空的容器记录 |
| 配送单 | 物料配送指令 |
| 工位库存 | 工位上的线边库存 |

## 业务流程

| 流程 | 说明 | 关联主要实体 |
|------|------|-------------|
| 工单管理 | 工单创建/编辑/状态管理 | 工单 |
| 工单下载 | 从ERP同步工单 | ERP工单、工单 |
| 维护工单工艺路线 | 调整工单的工艺路线 | 工单、工艺路线 |
| 上线 | 工单开始生产，条码扫描 | 工单、条码、过站记录 |
| 上料 | 按BOM将物料配送到工位 | 工单BOM、上料记录、工位库存 |
| 产品装配 | 执行装配操作并记录 | 产品装配记录、过站记录、工单上线数 |
| 下料 | 从工位取下完成品 | 下料记录、容器 |
| 下线 | 工单完成下线，更新完成数 | 工单完成数、已完工的条码 |
| 拉式倒冲 | 成品下线时自动扣减物料消耗 | 工单BOM、工位库存、上料记录 |
| 包材倒冲 | 包装材料自动扣减 | 工位库存、已完工的条码 |
| 包装 | 产品包装操作 | 包装关系、装箱单、产品等级 |

## 工单状态流转

```
pending ──→ released ──→ running ──→ completed
                               ↕         ↑
                              paused ────┘
                                 ↓
                             cancelled

说明：
  pending: 创建待排产
  released: 已发布可执行
  running: 生产中（可暂停/可完工）
  paused: 暂停（可恢复）
  completed: 全部完工（终态）
  cancelled: 取消（终态）
```

## 关键关系

```
工单 ──→ 工单BOM（物料需求清单）
工单 ──→ 工单工艺路线（执行工序序列）
工单 ──→ 制造计划（排产信息）
工单 ──→ 上料记录（物料消耗追溯）
工单 ──→ 下料记录（产出追溯）
工单 ──→ 产品装配记录（装配产出）
工单 ──→ 过站记录（每道工序通过记录）
```
