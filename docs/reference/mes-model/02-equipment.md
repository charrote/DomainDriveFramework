# 02 — 设备与资源管理

> **实体数：** 4 | **流程数：** 4  
> **覆盖：** 设备台账、工装夹具、测试设备、资源类型、工位

---

## 数据实体

| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 工装、夹具、测试设备 | 生产设备/工装/夹具/测试仪器台账 | 设备编码、设备名称、规格型号、设备类型、状态、所属产线、所在工位、采购日期、最近维保日期 |
| 资源类型 | 设备分类（生产设备/工装/夹具/测试设备等） | 类型编码、类型名称 |
| 工位 | 产线中的工位定义 | 工位编码、工位名称、所属产线、状态 |
| 工位库存 | 工位上的物料库存（线边库） | 工位、物料、数量、批次 |

---

## 业务流程

| 流程 | 说明 | 关联主要实体 |
|------|------|-------------|
| 工装、夹具、测试设备管理 | 设备台账维护 | 工装/夹具/测试设备、资源类型 |
| 工位维护 | 工位信息维护 | 工位 |
| 工位安全库存维护 | 设置工位物料的安全库存上下限 | 工位库存 |
| 资源层级维护 | 设备分类层级维护 | 资源类型 |
| 资源建模 | 资源属性模型定义 | 资源模型 |

---

## 关键关系

```
产线 ──→ 工位（一条产线有多个工位）
工位 ──→ 设备（一个工位可有多台设备）
设备 ──→ 资源类型（分类体系）
工位 ──→ 工位库存（一个工位有多种物料库存）

设备状态流转：idle → running → maintenance → idle
                              → fault → maintenance → idle
                              → offline
```
