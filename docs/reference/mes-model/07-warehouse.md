## 业务流程

### 入库流程
| 流程 | 说明 |
|------|------|
| 接收 | 物料到厂接收 |
| 检验交接 | 待检验物料与IQC交接 |
| 入库交接 | IQC合格后与仓库交接 |
| 上架 | 将物料放到指定货位 |
| 上架货位运算 | 自动计算推荐上架货位 |
| IQC报检 | 来料检验申请 |

### 出库流程
| 流程 | 说明 |
|------|------|
| 生成领料单 | 根据工单BOM生成领料需求 |
| 拣配 | 按领料单分配拣货任务 |
| 拣料 | 执行拣料操作 |
| 发料 | 将拣好的物料发给产线 |
| 退料 | 产线退回多余物料 |
| 退料标签打印 | 退料标签打印 |

### 库存管理
| 流程 | 说明 |
|------|------|
| 子库转移 | 仓库间/库区间物料转移 |
| 杂发 | 非生产性领料出库 |
| 杂收 | 非采购性入库 |
| 仓库盘点 | 执行盘点作业 |
| 财务抽盘 | 财务抽盘核对 |

| 盘盈 | 盘点盈余处理 |
| 盘亏 | 盘点亏损处理 |

### 退货
| 流程 | 说明 |
|------|------|
| 供应商退货 | 将不合格品退给供应商 |

## 仓库操作流程

```
【入库流程】
PO发布 → PO接收 → 来料接收 → IQC检验 
  → 合格 → 入库交接 → 上架货位运算 → 上架
  → 不合格 → 供应商退货

【出库流程】
生成领料单（从工单BOM触发）
  → 拣配（推荐拣货货位）
  → 拣料（扫描条码确认）
  → 发料（交接给产线）
  → 上料（投料到工位）

【退料流程】
产线剩余物料
  → 生成退料单
  → 退料标签打印
  → 退料交接
  → 仓库上架回库

【盘点流程】
生成/打印盘点单
  → 仓库盘点（初盘）
  → 财务抽盘（复盘）
  → 差异分析 → 盘盈/盘亏处理
```
