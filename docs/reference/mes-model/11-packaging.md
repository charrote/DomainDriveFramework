# 11 — 包装管理

> **实体数：** 6 | **流程数：** 4  
> **覆盖：** 包装操作、装箱管理、产品等级

---

## 数据实体

| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 包装属性 | 包装规格定义 | 包装编码、包装名称、容量、材质 |
| 包装关系 | 产品与包装的关联 | 产品条码、包装条码、包装层级 |
| 装箱单 | 装箱明细记录 | 装箱单号、产品列表、数量、毛重、净重 |
| 已包装的产品 | 已完成包装的产品记录 | 产品条码、包装条码、包装时间 |
| 产品等级 | 产品等级定义（一等品/二等品/合格品） | 等级编码、等级名称 |
| 产品分档结果 | 产品等级判定结果 | 产品、等级、判定人、时间 |
| 产品属性 | 产品特性属性（尺寸/重量/颜色等） | 产品、属性名、属性值 |

---

## 业务流程

| 流程 | 说明 | 关联主要实体 |
|------|------|-------------|
| 包装 | 产品包装操作（扫描关联） | 包装关系、已包装的产品 |
| 包装复核 | 包装内容核对 | 包装关系、装箱单 |
| 打印装箱单 | 装箱单打印输出 | 装箱单 |
| 等级判定 | 产品等级判定（OQC后） | 产品等级、产品分档结果 |

---

## 包装流程

```
下线产品 → 包装（扫描产品条码和包装条码）
        → 包装复核（核对数量）
        → 打印装箱单
        → 等级判定（如需要）
        → OQC检验 → 成品入库/出货
```
