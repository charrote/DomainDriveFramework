# 09 — 采购与供应商管理

> **实体数：** 6 | **流程数：** 3  
> **覆盖：** 采购订单、供应商管理、来料计划

---

## 数据实体

| 实体 | 说明 | 关键字段 |
|------|------|---------|
| 供应商 | 供应商主数据 | 供应商编码、供应商名称、状态、等级 |
| 供应商与物料关系 | 供应商可供应的物料清单 | 供应商、物料、价格、交期 |
| 批准供应商维护 | 已批准的供应商列表 | 供应商、物料、批准日期、有效期 |
| PO数据 | 采购订单 | PO单号、供应商、物料、数量、交期、状态 |
| 已发布的PO | 已发给供应商的PO | 关联PO数据 |
| 已接收的PO | 供应商已确认的PO | 关联PO数据 |
| 来料计划 | 物料到货计划 | 计划到货日期、物料、数量、PO |
| 采购分类 | 采购品类分类 | 分类编码、分类名称 |

---

## 业务流程

| 流程 | 说明 |
|------|------|
| 采购订单据维护 | PO的创建/编辑/跟踪 |
| 供应商数据维护 | 供应商主数据管理 |
| 供应商数据下载 | 从ERP同步供应商数据 |
| 批准供应商维护 | 供应商准入/剔除管理 |
| PO发布 | 将PO发布给供应商 |
| PO接收 | 供应商对PO的确认接收 |
| 来料计划维护 | 到货计划的维护 |
| 采购分类维护 | 采购品类的维护 |

---

## 与 WMS 的接口

```
供应商 ──→ PO数据 ──→ 到货接收（WMS）
                             │
                    来料计划 ──┘
                             │
                             ▼
                       IQC检验（品质）
                             │
                     合格 → 入库（WMS）
                     不合格 → 退货（供应商）
```
