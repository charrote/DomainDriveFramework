| 类型 | 代码 | 说明 | 触发时机 |

| 来料检验 | IQC | 物料入库前检验 | 接收→检验交接 |

## 业务流程

| 流程 | 说明 | 关联主要实体 |
|------|------|-------------|
| IQC报检 | 来料检验申请 | 报检单、抽样方案、检验项目 |
| 来料检验 | 执行来料检验并记录结果 | 检验记录、物料检验项目 |
| PQC | 制程质量检验 | 缺陷、产品缺陷记录、过站记录 |
| OQC检验 | 成品出货检验 | 已报检的产品、检验结果数据 |
| 复判 | 不合格品的再次判定 | 有缺陷的条码、已报废的条码 |
| 维修 | 不良品维修操作 | 产品维修记录 |
| 首件判定 | 首件检验与判定 | 首件检验单、首件判定结果 |
| 驻厂检验 | 供应商驻厂检验 | 资质报告、检验记录 |
| 8D管理 | 8D报告流程管理 | 缺陷、缺陷责任 |

## 检验流程

```
IQC（来料检验）：
  接收 → IQC报检 → 来料检验 → 合格 → 入库交接
                               → 不合格 → 退货/让步接收/返修

PQC（制程检验）：
  工序完成 → PQC抽检 → 合格 → 下一工序
                      → 不合格 → 复判 → 返修/报废/让步

OQC（成品检验）：
  包装完成 → OQC检验 → 合格 → 等级判定 → 入库/出货
                      → 不合格 → 复判 → 返修/降级/报废

首件检验：
  换线/换模/开班 → 首件检验 → 合格 → 批量生产
                            → 不合格 → 调整后复检
```

## 关键关系

```
检验项目 ←── 物料检验项目 ──→ 物料（一个物料对应多个检验项目）
检验项目 ←── 质量分类检验项目 ──→ 质量分类

报检单 ──→ 报检单明细 ──→ 检验项目
报检单 ──→ 抽样方案（决定抽检数量）

检验记录 ──→ 缺陷（发现哪些缺陷）
缺陷 ──→ 缺陷分类（三级分类）
缺陷 ──→ 缺陷责任（归属部门/岗位）

工单 ──→ 检验记录（工单维度的品质追溯）
产品 ──→ 产品缺陷记录（产品维度的缺陷追溯）
```
